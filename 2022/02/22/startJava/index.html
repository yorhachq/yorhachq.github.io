JDK&IDEA

关于JDK

　　JDK(Java Development Kit)，Java开发工具包，由Sun公司提供。其中包含了JRE(Java Runtime Environment)，Java运行环境。为便于教材学习跟进，这里选择与教材相同的JDK8(Java SE Development Kit 8u321)。若只使用JDK，不使用IDE，还应配置PATH环境变量。

关于IDEA

概述

　　IntelliJ IDEA，Java IDE之一，JetBrains公司的产品。旗舰版(Ultimate)收费，提供30天免费试用。社区版(Community)免费。当前最新版本为 2021.3.2(官网下载)，本文后续的Ultimate版本激活破解也仅针对2021.3.2。

旗舰版(Ultimate)激活破解

　　注意：社区版(Community)无需破解，本就开源，此处为旗舰版(Ultimate)激活破解，仅供个人学习使用。

　　请先前往官网下载并安装好旗舰版(Ultimate)，再执行后续步骤。

第一步：下载激活脚本

　　链接：https://pan.baidu.com/s/1LSVwWIk65rwMGYOaE8jfHg
　　提取码：fkrs

　　先通过网盘下载好激活补丁，打开文件夹如下：

  • scripts:放置了相关脚本，包含自动安装、卸载破解补丁脚本（Windows、Mac、Linux 对应系统的脚本都有）；

  • vmoptions:放置了 JetBrains 产品的 idea.vmoptions 配置文件。之前版本都是通过在这个文件中手动引入破解补丁，但是最新版本 IDEA 2021.3.2 官方加入了反制手段，在用户目录下已经找不到这个文件了，新版本我们直接引用这个文件夹下的 idea.vmoptions 配置文件；

第二步：运行激活脚本

　　将 ja-netfilter-all　激活文件夹移动到电脑上某个位置，请一直固定置于你选择的路径下，路径变更将导致激活破解失效。
　　

　　Windows 系统，运行 install-current-user.vbs 脚本，为当前用户安装破解补丁。

　　Mac/Linux 系统，运行 install.sh 脚本安装。

　　点击安装，会弹出如下提示：

　

　　告诉我们，运行此补丁大约花费几秒钟，点击 确定，等待 Done 完成提示框出现，到这里，表示补丁安装成功。

　

注意：运行此脚本因为需要添加 idea.vmoptions 文件的环境变量，可能会被杀毒软件误报为木马，大家允许运行即可。

第三步：打开 IDEA, 填入指定激活码完成激活

　　运行脚本安装破解补丁完成后，一定要重启 IDEA，然后，填入下面的激活码，点击激活即可。

　　复制激活码后填入，点击 Activate 按钮完成激活：
　

如果重启 IDEA 填入激活码依然无法激活，不妨重启下系统。

　　点击激活后，就可以看到激活到 2099 年了：
　

　

注意：激活成功后，请勿升级IDEA版本，可在设置中自行关闭
